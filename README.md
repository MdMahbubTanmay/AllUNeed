**All U Need** is a powerful automation tool designed to make setting up a new Windows environment effortless. Built with C# and WinForms, it provides a "hacker-style" interface to select and install over 40 essential applications using the **Windows Package Manager (Winget)**.

## 💡 Why I Made This?

Setting up a new PC or laptop is exciting, but the software setup is usually a headache. 

The idea for **All U Need** came to me when a friend bought a new laptop and spent hours searching for download links, dodging "fake" download buttons on third-party sites, and manually clicking "Next" on dozens of installers.

I built this tool to:
* **Save Time:** Turn a 2-hour manual process into a 1-click automated setup.
* **Ensure Safety:** Pull official packages directly from Microsoft’s Winget repository to avoid malware.
* **Simplify Life:** Help friends and fellow students at **UFTB** get their development environments ready in minutes.

---

## 🚀 Features

- **Batch Installation:** Choose from 40+ software items and install them in one go.
- **Silent Deployment:** Uses automated flags to accept licenses and run installs in the background.
- **Developer Aesthetic:** Clean, high-contrast dark mode UI.
- **Categorized Workflow:** Easily find tools for Browsers, Dev, Productivity, Media, and Security.

## ⚙️ How to Use

1. **Run as Admin:** Ensure you run the application with Administrator privileges so Winget can install software.
2. **Select Software:** Check the boxes for all the applications you need.
3. **Install:** Click the "Install" button. A command window will appear and handle the downloads and installations automatically.
